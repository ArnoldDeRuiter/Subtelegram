# Subtelegram
[![GitHub issues](https://img.shields.io/github/issues/WouterJanson/Subtelegram.svg)](https://github.com/WouterJanson/Subtelegram/issues)
[![license](https://img.shields.io/github/license/WouterJanson/Subtelegram.svg)](https://github.com/WouterJanson/Subtelegram/blob/master/LICENSE)

Subtelegram is a sweet and simple chat bot for Telegram, it returns the URL for a Subreddit when it is mentioned in a groupchat.
Very handy when you often refer to a Subreddit.

A live version is running at: [@SubredditLookupBot](https://telegram.me/SubredditLookupBot)

**Note:** Your message should be formated as `/r/subredditname` because of privacy reasons a Telegram bot can only see command messages by default.
